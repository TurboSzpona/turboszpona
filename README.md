## Czym jest TurboSzpona?

TurboSzpona bierze potężny silnik OpenClaw i pakuje go w **bezproblemową instalację**:

- **Jedna komenda** - `npm install -g turboszpona && turboszpona init`
- **Skonfigurowane proxy LLM** - Bez żonglowania kluczami API. Działa od razu.
- **Kreator osobowości** - Stwórz unikalną osobowość agenta w 8 krokach
- **Multi-kanałowy** - WhatsApp, Discord, Slack, Telegram, rozmowy telefoniczne
- **Natywna aplikacja macOS** - Piękny dashboard, bez terminala
- **Europejskie DNA** - Zgodność z RODO, pełne wsparcie polskiego

## Szybki start

```bash
npm install -g turboszpona
turboszpona init
```

To wszystko. Twój agent działa.
